# Pharmacy Website Project

This is a simple pharmacy website made with PHP and MySQL. It includes a basic admin panel to manage products.

## Main Features

- User sign-up and login
- View and buy pharmacy products
- Add products to cart
- Admin panel to add/edit/delete products

## How to Use

1. Import the `database.sql` file into your local server (like XAMPP or WAMP).
2. Open the project in your browser.
3. Go to `admin/login.php` to log in as admin.

Use the default admin credentials to log in. (You can change them in the database or configuration file.)


## Files Overview

- `index.php` – Homepage
- `shop.php` – Product listing
- `cart.php` – Shopping cart
- `admin/` – Admin panel files
- `assets/` – Images, CSS, etc.

That's it!
